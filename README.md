# Pelagea

Pelagea is a premium conceptual website for a fictional luxury marine species brand. The experience turns scrolling into a slow descent through ocean depth, moving from bright surface light to the near-black abyss.

## Pages

- Home
- About
- Journal
- Contact
- UX Case Study with EN / UA / PL / RU language switching

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Run Locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.
